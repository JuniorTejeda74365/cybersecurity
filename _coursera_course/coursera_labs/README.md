# Cybersecurity Projects Portfolio

This folder contains hands-on cybersecurity projects demonstrating incident response, network security analysis, risk assessment, and security auditing. These projects were completed as part of cybersecurity training and portfolio development.

Each report reflects real-world security scenarios and applies industry-standard frameworks and practices.

**File:** `incident_report_analysis.md`

- Analyzes a Denial-of-Service (DoS) attack caused by an ICMP flood
- Applies the NIST Cybersecurity Framework (Identify, Protect, Detect, Respond, Recover)

 ### 7. Algorithm for File Updates in Python
**Folder:** `algorithm_file_updates_python/`

- Developed a Python script to automate updates to an IP address allow list
- Reads IP addresses from a text file using file handling techniques
- Converts file contents into a list for processing
- Compares the allow list against a remove list using loops and conditional statements
- Removes unauthorized IP addresses and updates the original file
- Demonstrates Python automation for access control management
